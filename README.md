###로고

내 이름을 사용해서 잘 만들어 볼 수 있겠다 싶어 제작했다
영어로 바꾸면 성과 이름 모두 마지막에 in이 들어가는데
로고를 in과 관련시켜 사용하고
안에 내용도 in과 관련된 내용을 집어넣으면 좋겠다고 생각했다.
